About the project

This is a simple simulated world which is based on the style of the WOOD1 environment described from Wilson in "ZCS: A Zeroth Level Classifier System" (1994).

Agents can use TCP based socket connections with a plain text protocol to explore it. The protocol draft is defined in a Readme file in our github repository.

Also compare with the corresponding script on uffm.org.
Disclaimer: The author of the script is different from the authors of the implementations.
